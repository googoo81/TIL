## 데이터 타입의 종류

### 1. Number
- 정수(Integer) // 1, 2, 3, 4 ...
- 소수(Float) // 1.555, 2.545345 ...
- ※Number타입은 서로 연산기호를 이용하여 계산할 수 있다.

### 2. String
- String의 의미: 처음부터 끝까지 문자(Text)로 구성되어 있다는 의미

- "Hello " + " My name is googoo81" // Hello My name is gogoo81
- ※String타입은 ""을 이용하여 입력하고 String타입의 합은 두개를 합
쳐서 출력한다.
